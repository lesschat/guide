# guide
## python环境搭建

   1.安装brew[如果没有brew命令]

   `ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

   2.安装python

   `brew install python`

   3.安装pip

   定位到get-pip.py，运行命令

   `sudo python get-pip.py`

   参考:[python环境搭建](http://blog.csdn.net/fancylovejava/article/details/39140373)

## 启动环境

`mkdocs serve`


## 生成html文件

`mkdocs build`
