## OAuth2授权接口

|业务|url|描述|
|:-----------|:-----------|:----------|
|请求授权|/oauth2/authorize|用户请求授权
获取|access_token/oauth2/access_token|获取access_token
刷新|token/oauth2/refresh_token|根据refresh_token刷新获取最新access_token
